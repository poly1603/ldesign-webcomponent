💻 代码示例

// HTML <button id="myButton">点击我</button> <ldesign-drawer id="myDrawer" placement="bottom" anchor-mode="element" anchor-element="#myButton" anchor-mask-partial="true" drawer-title="向下展开的抽屉" > <div>抽屉内容</div> </ldesign-drawer> // JavaScript const button = document.getElementById('myButton'); const drawer = document.getElementById('myDrawer'); button.addEventListener('click', () => { drawer.open(); });
核心属性：
  • anchor-mode="element" - 启用锚点模式
  • anchor-element="#button" - 指定锚点元素
  • anchor-mask-partial="true" - 启用部分遮罩
  • placement - 展开方向（top/bottom/left/right）

🎯 向下展开的抽屉

这个抽屉从按钮下方展开，只在下方区域有遮罩，上方区域可以正常交互。

特点：
  • 从触发按钮位置展开
  • 只在展开方向添加遮罩
  • 未展开区域可以正常点击
  • 适合下拉菜单、筛选器等场景
